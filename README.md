# Redux Under The Hood 🧠

Цей репозиторій створено для глибокого розуміння внутрішньої архітектури **Redux**. Замість використання готової бібліотеки, тут реалізовано власну версію системи керування станом з нуля на **TypeScript**.

## 🚀 Мета проєкту
- Зрозуміти, як працює патерн `Flux` та архітектура Redux.
- Вручну написати функцію `createStore`.
- Реалізувати методи `.getState()`, `.dispatch()` та систему підписок `.subscribe()`.
- Розібратися з чистими функціями (`reducers`) та незмінністю стану (`immutability`).

## 🛠 Технологічний стек
- TypeScript
- Node.js (для запуску та тестування в консолі)

## 📁 Структура проєкту
- `src/core/redux.ts` — власна реалізація методу `createStore`.
- `src/store/` — редюсери, екшени та конфігурація стору.
- `src/index.ts` — точка входу для перевірки роботи (dispatch екшенів, логування стану).
