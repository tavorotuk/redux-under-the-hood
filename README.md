# Redux Under The Hood 🧠

Цей репозиторій створено для глибокого розуміння внутрішньої архітектури **Redux**. Замість використання готової бібліотеки, тут реалізовано власну версію системи керування станом з нуля на **JavaScript (ES Modules)**.

## 🚀 Мета проєкту
- Зрозуміти, як працює патерн `Flux` та архітектура Redux.
- Вручну написати функцію `createStore`.
- Реалізувати допоміжні функції `combineReducers` та спрощену версію `createSlice` (аналог з Redux Toolkit).
- Реалізувати методи `.getState()`, `.dispatch()` та систему підписок `.subscribe()`.
- Розібратися з чистими функціями (`reducers`) та незмінністю стану (`immutability`).

## 🛠 Технологічний стек
- JavaScript (ES Modules)
- HTML5 / CSS3 (для демонстраційного інтерфейсу)

## 📁 Структура проєкту
- `redux.js` — власна реалізація функцій `createStore`, `combineReducers` та `createSlice`.
- `store/` — редюсери, екшени та конфігурація стору:
  - `store/index.js` — ініціалізація та експорт стору.
  - `store/amount.js` — слайс для керування сумою (`amount`).
  - `store/goods.js` — слайс для керування списком товарів (`goods`).
  - `store/position.js` — слайс для керування координатами (`position`).
- `app.js` — точка входу, яка демонструє роботу стору (підписка, dispatch екшенів та виведення стану в консоль).
- `index.html` та `style.css` — сторінка та стилі для відображення інтерфейсу.

## 🚀 Як запустити
Оскільки проєкт використовує ES-модулі в браузері, відкрийте `index.html` через будь-який локальний веб-сервер (наприклад, Live Server у VS Code) та відкрийте консоль розробника (DevTools) для перегляду логів стану.
